What is it

DM CHM Decompiler is a small little add-on I made in Visual Basic 6 for Windows and will allow you to decompile CHM (HTML Help Files) I made
this program because I needed a way to get back my files so I found a useful little command under DOS and decided to make a program that can be used in windows,
